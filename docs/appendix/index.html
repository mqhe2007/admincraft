# 默认布局

meta.layout 值: default

默认为控制台布局，是系统主要布局，包括头部，侧边菜单，尾部，页签栏，模块页面出口等。此布局用于绝大多数业务模块路由。

# 毛坯布局

meta.layout 值: blank

毛坯布局为全空白页面，任何路由页面选用此布局后，只会显示路由页面自身内容。常用于系统注册流程，登录页，帮助页面等业务外围页面。
