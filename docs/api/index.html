# API

# 全局 API

# Admincraft.use

  • # 参数:

    • {Object | Function} plugin

  • # 用法:

    安装 Vue.js 插件。此处仅仅是起到了转发参数的作用，实际内部使用的还是 Vue.use 接口。

    该方法需要在调用 new Admincraft() 之前被调用。

// 虚拟的例子
import vuePlugin from 'vue-plugin'

Admincraft.use(vuePlugin)
let app = new Admincraft({})

# 实例 API

# app.$addRoutes

  • # 参数:

    • {Array} routes

    • {Function} callback

  • # 用法:

    添加路由配置

this.$addRoutes(routes, () => {})

# app.$addStore

  • # 参数:

    • {String} vuexModuleName

    • {Object} vuexModule

    • {Function} callback

  • # 用法:

    添加 Vuex 状态模块

this.$addStore('vuexModuleName', vuexModule, () => {})

# app.$addMenus

  • # 参数:

    • {Array} routes | menus

    • {Function} callback

  • # 用法:

    为侧边栏菜单添加菜单项，可以直接传入路由配置数组，也可以传入菜单配置数组数组。

this.$addMenus(someArray, () => {})

# app.$addDynamicComponent

  • # 参数:

    • {String} position

    • {Object} component

    • {Function} callback

  • # 用法:

    为预设的动态组件区域添加组件

this.$addDynamicComponent('HEADER_RIGHT_MENUS', Component, () => {})

# app.$removeDynamicComponent

  • # 参数:

    • {String} position

    • {String} componentName

    • {Function} callback

  • # 用法:

    移除某个动态组件区域内的某个组件

this.$removeDynamicComponent('HEADER_RIGHT_MENUS', 'componentName', () => {})

# app.$addRemoteLib

  • # 参数:

    • {Array} libUrl

  • # 用法:

    添加远程库，因为远程资源加载是异步过程，所以方法返回一个 Promise。

this.$addRemoteLib(libUrlArray).then()

# app.$modifyHomepage

  • # 参数:

    • {String} routeName

    • {Function} callback

  • # 用法:

    改变首页。

this.$modifyHomepage('routeName', () => {})

# 实例 API / eventBus

事件总线通过 app.$eventBus 上的若干个方法进行调用，具体说明如下：

# $eventBus.on()

  • # 参数:

    • {String} eventName

    • {Function} callback

监听一个事件，并且指定回调函数。

# $eventBus.emit()

  • # 参数:

    • {String} eventName

    • {any} payload

触发一个事件，并且发送任意数据作为监听方法回调函数的参数。

# $eventBus.off()

  • # 参数:

    • {String} eventName

取消一个事件的监听，也就是清空事件回调队列里的所有函数。

# $eventBus.clean()

  • # 参数: 无

清空 eventBus，这会取消所有事件监听。
