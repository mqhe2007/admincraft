Admincraft

一个支持模块化分布部署和加载的 Vue Admin

快速上手 →

一切基于Vue.js生态

Admincraft 基于 vue 生态的最佳方案（希望），内置了常用的Vue周边库，并且提供了丰富的配置项让您自由配置。

可选高质量UI组件库

Admincraft 可选安装整套适用于中后台系统的UI组件库和布局页面，开箱即用。但是 Admincraft 不会强制您使用它们，您可以使用自己喜欢的其他UI组件库。

模块化开发与加载

Admincraft 提供了模块化开发和加载的方法。借助于 webpack ，可以把代码打包成能分布部署的“模块”；模块本身又能在任何时候加载另一个模块。使用了 Admincraft 的项目看起来就像是一节节火车车厢，首尾相连就成了一列巨大的列车。

# 使用简单

# 1. 安装 admincraft

yarn add admincraft

# 2. 导入 admincraft

import Admincraft from 'admincraft'

# 3. 创建实例

let admincraft = new Admincraft()
admincraft.$mount('#app')
